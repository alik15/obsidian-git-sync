## Zip

 ```bash
 zip <archivename.zip> <filename>
```
### Installing
Install the utility
```bash
sudo apt-get install unzip
```

### Unzipping

unzips to a folder
```bash 
unzip file.zip -d destination_folder
```

unzips with same name in current working directory

```bash
unzip file.zip
```
